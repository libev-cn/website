    文件命名规则

    一般情况下, Libev的源码打包后的文件命名规则是以Libev-version`.tar.gz`形式出现.

    获取源码包

    译者推荐大家从作者提供的文件服务器上下载源码包, 但是鉴于国际网络互通的一些问题有些同学可能无法访问.
